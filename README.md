# Machine Learning-Based Detection of AI-Generated Academic Submissions in Nigerian Tertiary Institutions

## Overview
This project investigates the use of Machine Learning and Neural Networks to detect AI-generated academic submissions in Nigerian tertiary institutions.

The system compares human-written academic texts with AI-generated content produced by ChatGPT, Claude, and Gemini.

---

## Objectives
- Detect AI-generated academic writing
- Compare Machine Learning models
- Evaluate Neural Network performance
- Explore AI-powered academic integrity systems

---

## Dataset
The dataset consists of:
- Human-written essays and assignments
- AI-generated texts from ChatGPT
- AI-generated texts from Claude
- AI-generated texts from Gemini

---

## Methodology
### NLP Preprocessing
- Lowercasing
- Tokenization
- Stopword removal
- Lemmatization

### Feature Extraction
- TF-IDF Vectorization

### Models Used
- Logistic Regression
- Random Forest
- Neural Network (TensorFlow/Keras)

---

## Results

| Model | Accuracy |
|---|---|
| Logistic Regression | 73% |
| Random Forest | 81% |
| Neural Network | 81.8% |
